# Getting started

## Create the virtual environment
```
pyenv virtualenv 3.4.4 webspeak-3.4.0
```

## Activate the virtual environment

```
pyenv activate webspeak-3.4.0
```

## Uninstall all pip packages

```
pip install -r requirements.txt
```
